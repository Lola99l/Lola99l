Logical Operators

These operators combine conditional statments, returns either true or false.

Python Operator Example Description
and a > 3 and a < 10 Returs true if Both conditional statments are met. False otherwise.
or a < 3 or a > 10 Returns true if one or more of the conditional statments are met. False only of both atatments arent met.
not not(a > 3 and a < 10) Reverses the result of the conditional statments; returns false if the result is true, and vice verca.

Membership Operators

These operators are mainly used with objects, used to check if a sequence if present in the object.

Python Operator Example Description
in a in b Returns True if a sequence with the specified value is present in the object
not in a not in b Returns True if a sequence with the specified value is not present in the object

Identity Operators

These operators are used to compare said objects, to compare weather they are the same object with the same memory location.

Python Operator Example Description
is a is b Returns True if both variables are the same object
is not a is not b Returns True if both variables are not the same object
