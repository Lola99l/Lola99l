Libraries

Python is an open srouce programming language, which is a very good thing for creating Libraries . So what are a Library ?. A library is a collection of related modules. It contains bundles of code that can be used repeatedly in different programs to support our case.

Importing Libraries

To use the collection of codes (Library), First you will need to import it. There are many ways to import Libraries the most common way is to import the Library directly to the enviroment of the compiler, As the example below.

Example on importing the pandas library
Import Pandas as pd # We import the library then name it as pd
df = pd.read('file.csv') # we read the file and store it in a variable named df
print(df.to_string()) # Then print the file

Importing Items:

In Python you can import specifc items or functions inside any library to use it in your code insted of writing it by your self, As we say work smarter not harder.

Importing Functions from a library is also better than importing all the library to use one function.

Example
from math import sqrt, sin # Get sqrt from math
A = 16
B = 3.14
print(sqrt(A)) # use sqrt function
print(sin(B)) # use sin function
