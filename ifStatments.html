What are If Statments?

If Statments are declared in ones programe with the use of conditional statements to determine witch of the actions specified by the user is going to put in play. The conditional statments are the same ones mentiond in the Operators section of the website.

Basic If Statment

When declaring the "If statments" the if keyword must be used.

Example
a = 200
b = 300

if a < b:
# The print statement will only run if the condition above is met
print("b is greater than a")

Else:

The else statement will act as a sort of default; if all the preceding if statments were not met, the action or value we have given the else statment will be taken.

Example
a = 100
b = 50
if b > a:
print("b is greater than a")
else:
print("b is not greater than a")
