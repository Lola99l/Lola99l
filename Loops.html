Welcome to python Loops

In pyhton we have two primative types of loops:

  • While Loops
  • For Loops

While Loops

A while loop will execute a statment or multiple statements as long as the given condition is met.

Example
i = 0
while i < 10:
print(i)
i += 1

For Loops

A for loop is used for iterating over a sequence, unlike other programming languages the for keyword here works more like an iterator method as found in other object-orientated programming languages.
With the For loop we can execute a set of statements, once for each item in a list, tuple, set.

Example: Print all the subjects in the given list.
Subjects = ["Science", "Math", "English","IT"]
for x in Subjects:
print(x)

Different Loops/Functions:


Break Statement Continue Statement Range Function Nested Loops

The Break Statement:

With the break statement we can stop the loop before it has looped through all the items.------------------------------------------------------------------------------

Example
Subjects = ["Science", "Math", "English","IT"]
for x in Subjects:
print(x)
if x == "Math":
break

The Continue Statement:

With the continue statement we can stop the current iteration of the loop, and continue with the next: ------------------------------------------------------------------------------

Example
Subjects = ["Science", "Math", "English","IT"]
for x in Subjects:
print(x)
if x == "Math":
continue

Range Function:

To loop through a set of code a specified number of times, we can use the range() function, The range() function returns a sequence of numbers, starting from 0 by default, and increments by 1 (by default), and ends at a specified number.

Example
for x in range(6):
print(x)
# Here the values are not 0 - 6 but they are 0 - 5

# We can also set a starting point fro the loop:
for i in range(2,10):
print(i)

Nested Loops:

A nested loop is a loop inside a loop.
The "inner loop" will be executed one time for each iteration of the "outer loop": ------------------------------------------------------------------------------

Example
grade = ["97", "89", "95", "99"]
subj = ["Science", "Math", "English","IT"]
for i in grade:
for j in subj:
print( i , j )
