What is a variable?

A variable in python is a symbolic name that acts as some sort of pointer.
Once an object is assigined to a variable, that same variable can be reffered to throughout the rest of the implemented code by refrencing the pointer name.

Decleration

Declaring a variable in python differes from other programming languages, in such a way that you do not need to specify the variable type.

Example
a = 5
b = "python"

# Variable types can be directly changed even after the have been set.

s = 12 # Here s is of type int
s = "Welcome" # Now s is of type str

Different Properties:


Variables in python have certain properties that affect the way they are perceived by both the humman brain and the compiler itself.


Casting Type Obtaining Quotation Case-Sensitivity

Variable Casting:

When you are in need of specifying the data type of a certain variable for any which reason, you will use a concept known as Casting.
Casting if the variable will affect how it is saved in the program which in hand affects the way the variable could be used.

In the three variable declaraed in the Example, all were given the same value of 3; but due to the casting we have implemented the way they are precieved differs.

Example
X = int(3) # X will be saved as 3
Y = float(3) # Y will be saved as 3.0
Z = str(3) # Z will be saved as '3'

Type Obtaining:

After we declare any variable in the code program, we have the capability of figuring out the type of the variable in case we were unsure of it or to check if there was any casting used on the variable. This can be done via using the type() function.

Example
a = 5
b = "Hello World"
print(type(a)) # The resulting output would be int
print(type(b)) # The resulting output would be str

Quotations:

Unlike the syntax of other programming languages, python doesnt differentiate between a single quotation and double quotation declared variables. As they are all seen as the same type of variable, they can be concatinated together without any errors emerging.

Both the declarations in the example shown are the exact same

Example
p = 'Hello World'
# Here the variable is declared using single quotations
p = "Hello World"
# Here the variable is declared using double quotations

Case Senssitivity:

Python variable names are case-sensitive; if the names differ no matter how miniscule the difference is they will save different values, these small differences will create new variables.

Example
c = 4
C = "python"
# In this case, C doesnt overwrite c; they are different variables.
